# Assinatura de E-mail Profissional em HTML

Este é um projeto simples que desenvolvi para aplicar meus conhecimentos básicos de **HTML e CSS**, criando uma assinatura personalizada para os meus e-mails profissionais.

## Tecnologias utilizadas:
* **HTML5**: Estruturação da assinatura.
* **CSS Inline**: Estilização de cores e fontes para compatibilidade com clientes de e-mail (Gmail/Outlook).

## Objetivo:
Melhorar minha apresentação profissional enquanto estudo os fundamentos da Web, com foco futuro em **Desenvolvimento Backend**.
